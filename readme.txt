== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.


1. FTP programınız yada plugin yükleme aracınız ile eklentiyi yi wp-content/plugins dizinine atın.
2. Eklentiyi etkinleştirin.
3. Eklenti editörü ile YIA yı açın, $r[] değişkenlerine kendi reklamlarınızı tanımlayın. Daha fazla reklam eklemek isterseniz 
$r'lerin başındaki # işaretlerini silin ve yukarıda örnekte gördüğünüz şekilde reklam kodlarınızı tırnaklar arasına ekleyin.

Bu eklediğiniz kodun içerisinde tek tırnak ( ' ) var ise tırnak önüne ters slash koyun ( \')

4.Hata oluşmasına karşı reklam eklerken kesinlikle yia.php nin yedeğini alın !
5.Hepsi bu kadar. Yazılarınızın içerisinde [yia id] şeklinde reklamarınızı eklemeye başlayabilirsiniz.
